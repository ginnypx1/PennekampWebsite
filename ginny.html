App & Web Development

Creative, curious and motivated iOS app developer with over 10 years experience working in large-scale, high-pressure storytelling, project management and marketing/media capacities. My goal is to create and maintain mobile apps that promote exploration and curiosity with inventive and fun UIs. I love to be challenged – I’m passionate about learning new tech and new techniques. Dependable and detail-oriented, I create strong, readable and maintainable source code.

  • Technical Skills
  • Languages: Swift, Python, HTML/CSS, JQuery, JavaScript, node.js, JSON, Objective C and SQL
  • Development Software: XCode, Sketch, Sublime Text, Git and GitHub
  • iOS Libraries: Animation, AutoLayout, AVFoundation, Cocoa, Cocoa Pods, MapKit, UIKit, URLSession, Unit Tests
  • Web Development Frameworks: Kitura, Django, Flask and Bootstrap
  • Databases: Core Data, Firebase, Parse, SQLite, MySQL
